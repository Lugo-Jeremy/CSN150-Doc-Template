## Steps I followed
1. First step I did was download Arduino IDE for Windows.
2. Once the install wizard finished, I opened the application and downloaded all the drivers
3. Next I plugged in my ESP32Cam with a USB-C adapter cable and the power LED lit up a red color LED.
4. Next, I looked up the link I posted above to setup the ESP32Cam in Arduino
5. I had to install the ESP32Cam boards to set it up, going through the process of opening File in the top left and opening preferences to paste a URL link in the "Additional boards manager URL" bar
6. Next I looked up how to set up a "Wi-Fi Scanner" and it was a source of code that needed to be executed. I copy and pasted the code into Arduino to be executed into the ESP32 Cam
7. I verified the code and it was good to execute
